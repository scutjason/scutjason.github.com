海森矩阵

发表于 2019-05-20 | 分类于 machine learning | 阅读数

本文介绍机器学习中一个重要的矩阵-海森矩阵

海森矩阵的定义

海森矩阵是一个多元函数的二阶偏导数构成的方阵，描述了函数的局部曲率。
海森矩阵常用于牛顿法解决优化问题，利用黑塞矩阵可判定多元函数的极值问题。
在工程实际问题的优化设计中，所列的目标函数往往很复杂，为了使问题简化，
常常将目标函数在某点邻域展开成泰勒多项式来逼近原函数，此时函数在某点泰勒展开式的矩阵形式中会涉及到海森矩阵。

海森矩阵的性质

通常用于求多元函数的极值点。
比如三元函数$f(x,y,z)$在(x_i,y_i,z_i)点的偏导数等于0(可能是极值点)。
$\frac{∂f}{∂x}|x_i=0, \frac{∂f}{∂y}|y_i=0, \frac{∂f}{∂z}|z_i=0$
然后求$f(x,y,z)$在(x_i,y_i,z_i)点的海森矩阵A。
1、当海森矩阵A是正定矩阵时，那么在该点处有极小值；
2、当海森矩阵A是负定矩阵时，那么在该点处有极大值；
3、当海森矩阵A是半正定矩阵或者半负定时，那么在该点处有可能有极值点，不确定，通过其他方法判断。
4、当海森矩阵A是不定矩阵时，那么在该点处不是极值点。
