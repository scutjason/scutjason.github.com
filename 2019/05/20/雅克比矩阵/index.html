雅克比矩阵

发表于 2019-05-20 | 分类于 machine learning | 阅读数

本文介绍机器学习中一个重要的矩阵-雅克比矩阵

雅克比矩阵的定义

向量微积分中，雅可比矩阵是一阶偏导数以一定方式排列成的矩阵，其行列式称为雅可比行列式。
雅可比矩阵的重要性在于它体现了一个可微方程与给出点的最优线性逼近。因此，雅可比矩阵类似于多元函数的导数。

对于$R^n→R^m$的函数$\vec{y}$(也就是这个函数的值是一个n维向量，不是一个简单的值),它对各个自变量x的偏导数，组成了m*n的矩阵

雅克比矩阵的性质

雅克比矩阵它表现了一个多变量向量函数的在某个可微点最佳线性逼近。
因此雅可比矩阵的行列式可用于求解点A的微分方程组的近似解。
我们知道，矩阵的行列式的值的几何意义：矩阵对应的线性变换前后的面积比。
而雅克比矩阵它实际上是用微分来近似表示在某个点的线性变换。
